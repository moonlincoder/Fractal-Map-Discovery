Mouse
-  **Wheel Up** - increase image quality
-  **Wheel Down** - reduce image quality
-  **Left Click** - scope x5 to clicked point
-  **Right Click** - move away from the point (also x5)

Keyboard
-  **I** - insert coords using command line
